Airtest开源项目
================

自动化测试框架
^^^^^^^^^^^^^^^

*   基于图像匹配，适合做游戏的自动化测试，同样也可以用作App
*   支持Windows/Android/iOS
*   提供[IDE](http://init.nie.netease.com/airtest_home/)，0上手成本，支持完整的工作流程：录制、回放、生成报告
*   同样作为Python库，可以用作其他测试框架的底层，提供跨平台的模拟操作、断言、报告生成等功能
