airtest.aircv.smart_crop module¶

本文件用于智能monkey，自动检测画面中可点击的控件： 控件分类：

1、文字按键类型（商城-背包-聊天框） 2、完整轮廓（比如按钮-图标-人物头像） 3、特殊角点（箭头）
方法1：
integral_preprocess(gray_img):
Laplacian → 二值化 → 膨胀-膨胀 → 提取轮廓 → 轮廓筛选 适用于UI界面的主要控件提取，细节提取较为不足。
方法2：
detail_detect(img):
sobel → 二值化 → 膨胀-腐蚀-膨胀 → 提取轮廓 → 轮廓筛选 默认使用Sobel预处理：适用于各种界面，细节完备，但是提取点太多。
